
Latest version:
Compile and link with
$ g++ -Wall crts.cpp -o crts -lm -lliquid -lpthread -lconfig -luhd -lliquidusrp

All scenario config files should be placed in the scconfigs/ folder.
All cognitive engine config files should be place in the ceconfigs/ folder.


To compile crts_rx.cpp:
$ g++ -Wall crts_rx.cpp -o crts_rx -lm -lliquid -lpthread -lconfig -luhd -lliquidusrp
